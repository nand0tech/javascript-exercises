# JavaScript Exercícios

Coleção de exercícios básicos de JavaScript para praticar lógica de programação.

## Exercícios

- Média de aluno
- Contador de cliques
- Par ou ímpar
- Maior número
- Calculadora
- Tabuada
- Fatorial

## Tecnologias

- HTML
- CSS
- JavaScript

## Objetivo

Praticar lógica de programação e manipulação do DOM.
